🎯 How To Play - Solo Mode

×

🎮 Objective

Test your memory by repeating increasingly complex color sequences. See how far you can go!

📋 Rules

  • Watch the sequence of colors that light up
  • Repeat the sequence by clicking the buttons in the same order
  • Each round adds one more color to the sequence
  • Make a mistake and the game ends
  • Your high score is saved automatically

🏆 Scoring

Your score increases with each successful round. Challenge yourself to beat your high score!

⚔️ How To Play - 2 Players Duel

×

🎮 Objective

Battle head-to-head in turn-based memory warfare! Each turn adds a new color to the sequence. Survive longer than your opponent!

📋 Rules

  • Turn Order: Player 1 goes first, then Player 2
  • Player 1's Turn: A new color is added to the sequence, then Player 1 must repeat the entire sequence in exact order
  • Player 2's Turn: Another new color is added, then Player 2 must repeat the entire sequence in exact order
  • Growing Sequence: Each turn adds one more color, making the sequence progressively longer
  • Elimination: Make a mistake and you're eliminated. No second chances!
  • Victory: Last player standing wins the duel

🏆 Victory

Outlast your opponent by maintaining perfect accuracy. One mistake and you're eliminated!

🔺 How To Play - 3 Players Battle

×

🎮 Objective

Triple threat memory battle! Each player's turn adds a new color to the growing sequence. Outlast both opponents to claim victory!

📋 Rules

  • Turn Order: Player 1 → Player 2 → Player 3, then repeat
  • Each Turn: A new color is added to the sequence, then the current player repeats the entire sequence in exact order
  • Round 1: P1 gets 1 color, P2 gets 2 colors, P3 gets 3 colors
  • Round 2: P1 gets 4 colors, P2 gets 5 colors, P3 gets 6 colors (if all survive)
  • Elimination: One mistake eliminates you permanently
  • Victory: Last player standing wins the triple threat!

🏆 Victory

Be the last player standing in this intense three-way memory battle!

💀 How To Play - Fatal 4-Way

×

🎮 Objective

Four-way memory mayhem! Each turn adds complexity as players build an ever-growing sequence. Survive the chaos to emerge victorious!

📋 Rules

  • Turn Order: Player 1 → Player 2 → Player 3 → Player 4, then repeat
  • Each Turn: A new color is added to the sequence, then the current player repeats the entire sequence in exact order
  • Round 1: P1 gets 1 color, P2 gets 2 colors, P3 gets 3 colors, P4 gets 4 colors
  • Intensifying Pressure: By Round 2, sequences start at 5+ colors
  • Fatal Mistake: One wrong move and you're eliminated forever
  • Survival of the Fittest: Only the strongest memory survives this 4-way battle

🏆 Victory

Survive the chaos and prove you have the ultimate memory. Fatal 4-Way champion!

🌟 How To Play - Pentagonal Panic

×

🎮 Objective

Five-player memory marathon! Each turn builds an increasingly complex sequence. Conquer the longest wait times and ultimate pressure!

📋 Rules

  • Turn Order: Player 1 → Player 2 → Player 3 → Player 4 → Player 5, then repeat
  • Each Turn: A new color is added to the sequence, then the current player repeats the entire sequence in exact order
  • Round 1: P1 gets 1 color, P2 gets 2 colors... P5 gets 5 colors
  • Round 2: P1 gets 6 colors, P2 gets 7 colors... and so on
  • Maximum Chaos: Longer wait times between turns add psychological pressure
  • Ultimate Challenge: Only the most focused mind can survive this 5-player gauntlet

🏆 Victory

Achieve legendary status by conquering the ultimate 5-player memory mayhem!
